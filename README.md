# AI for Business Leaders

A practical guide to understanding and leveraging artificial intelligence for non-technical business leaders.

## About This Book

This book provides business leaders with the knowledge and frameworks needed to make informed decisions about AI without requiring technical expertise. It covers:

- **Part I: Understanding What AI Really Is** - Mental models for AI capabilities and limitations
- **Part II: Using AI in Everyday Work** - Practical applications for daily tasks
- **Part III: Building Real AI Projects** - Implementation, measurement, and scaling

## Building the Book

### Prerequisites

- LaTeX distribution (TeX Live, MiKTeX, or MacTeX)
- The `memoir` document class

### Compilation

```bash
cd book
pdflatex main.tex
pdflatex main.tex  # Run twice for TOC and references
```

## Target Audience

- Executives evaluating AI investments

- Non-technical approach (no coding or math required)
- Practical prompt templates and examples
- Decision frameworks for common AI choices

- Security and compliance guidance
- Measurement and ROI frameworks
